Ejercicio 1

Creando una página web

  • 1. Descargar TextMate.
  • 2. Crear un usuario en Github.com y crear un repositorio. Copiar el link del repositorio en una nueva pestaña de chrome.
  • 3. Ir a html5up.net y descargar una plantilla con sub-páginas.
  • 4. Abrir el archivo index.html en TextMate y comenzar a personalizar la página principal, para monitorear los cambios, abrir el archivo index.html en chrome y refrescar tras cada modificación.
  • 5. En GitHub, dentro del repositorio creado, ir a la pestaña de "Code" y subir todos los archivos de la plantilla. Seleccionar cada uno, no subir la carpeta entera.
  • 6. Al refrescar el link del repositorio, los cambios hechos a la plantilla en el index.html deberían aparecer.
  • 7. Para modificar y personalizar las sub-paginas, abrir y editar el respectivo archivo de html.
  • 8. Cada que se edite un archivo de html en TextMate, volver a subir a GitHub y refrescar el link del repositorio.
