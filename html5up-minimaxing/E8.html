Ejercicio 8

Descargando datos de internet.

Para pasar los datos de una página de internet a un base de datos sin tener que hacerlo a mano, se puede utilizar la fórmula =importhtml en GoogleSheets. Para utilizar esta fórmula sólo hace falta especificar de qué página se quiere extraer la información, en qué formato está desplegada en la misma, su posición y en qué idioma está. Para evitar que la fórmula quede muy larga al insertar el URL completo de la página, este se puede pegar en una celda aparte y en la fórmula seleccionar la celda. La fórmula debería verse algo así:

En caso de que se quiera descargar una serie de datos desplegados de manera similar en páginas diferentes sin tener que repetir el proceso a mano, es posible sistematizarlo de la siguiente forma:

  • 1. Con el botón derecho, inspeccionar la página de dónde se quieren extraer los datos.
  • 2. Identificar las secciones del código html que corresponden a los sitios de donde se quiere extraer la información.
  • 3. Copiar las secciones identificadas a un editor de texto como TextMate.
  • 4. Limpiar el código borrando las partes que se repiten utilizando la herramienta de buscar y remplazar.
  • 5. Una vez limpiado, descargar el documento como csv y abrirlo en Excel.
  • 6. Unir lo obtenido con la limpieza del código con el inicio del URL para conseguir los URLs individuales de todas las páginas de dónde se quieren extraer los datos.
  • 7. Copiar los URLs obtenidos a una páina de GoogleSheets y utilizar la fórmula =importhtml para descargar los datos deseados.
  • 8. Cada que se edite un archivo de html en TextMate, volver a subir a GitHub y refrescar el link del repositorio.
